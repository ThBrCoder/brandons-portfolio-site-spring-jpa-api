Instructions

The REST API can be accessed using Postman through: www.brandontheisen.com/api/games


This is a simple Games API which exposes a list of some Nintendo 64 games. Supports GET, POST, PUT and DELETE HTTP requests. Only 10 games can be stored in the API list at a time, and the game's title and year must be valid.


Click here to view the current games stored in the API list.

The full supported JSON format for Game requests is: { "id" : int, "title" : "string", "year" : int }


When performing a PUT request, you can change the game to another title. The title and year must be correct according to the list above, otherwise the change will not succeed.
